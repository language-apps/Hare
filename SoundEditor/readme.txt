To get help to work first update the helpData directory. Specifically, files listed below.
Be very careful, the programs give up with no error messages when the syntax is wrong.

Each folder with the help set files contains
1) acorns.hs - definition of the help set views
2) map.jhm - definition of target weg pages
3) index.xml and toc.xml - Definition of target and index views
4) jhindexer.jar - Jar files to create search view

To Create search view
2) Run the makeSearchView.bat file to create the JavaHelpSearch directory
3) To Test: run testSearchView.bat 
4) Type search queries and, when done, type . to exit
